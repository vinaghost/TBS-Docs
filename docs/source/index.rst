Welcome
====

**TravianBotSharp** is a **free** Travian bot support `Travian Official <https://www.travian.com>`_, `TTwars <https://ttwars.com>`_ (and more if you donate)

Main features
====

* Multi-account support
* Auto builder (support use resource from hero inventory)
* Auto send hero to adventures
* Auto send farmlist 
* Auto improve troop
* Auto NPC

.. note::

	This project is under active development.

Known issue
====

* TBS only works with English language. Please change your language to English before using TBS.
* Cannot login same connection to multiple server (including non-proxy connection)
* Sometime TBS's timer is died ( task keep waiting ), you need to close and reopen TBS to fix it.
